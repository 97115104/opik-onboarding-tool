# Opik overview (slide source)

The onboarding UI Overview step uses the typed slide deck in
`apps/onboarding-ui/src/content/overviewSlides.ts`. Keep this file aligned with those slides.

## Slides

1. **What Opik is**: Open-source platform from Comet for building, evaluating, and operating generative AI apps.
2. **The problem it solves**: Full traces for multi-step LLM workflows that traditional logs fragment.
3. **Core capabilities**: Observability, evaluation, production monitoring, optimization and safety.
4. **Who Opik is for**: Builders, PMs, support, and businesses (personal and team value).
5. **Deployment options**: Self-hosted (`./opik.sh`), Comet Cloud, Enterprise. This wizard uses self-hosted.
6. **How this wizard uses Opik**: Local stack, chat demo trace, learn, pick an issue, draft PR with Cursor.
