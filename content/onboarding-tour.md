# Opik onboarding tour (structured UI)

The interactive checklist lives in `TourStep.tsx` (`data-testid="step-tour"`).

Guided actions:

1. Open Opik UI
2. Send a chat demo message
3. Open the new trace and check Metadata and Token usage (duration is secondary)

Troubleshooting for down services is handled by Local stack auto-heal / Fix this.
